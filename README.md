# Pololib
This project is on hold
## Description
Have you ever noticed whats wrong with comercial engines? One could say they offer a great variety of tools to help thousands of people create their games, but I bet that most of those users havent used a fraction of those features. This can overstimulate new users or tank performance for some games.
Although not the ultimate solution, Pololib aims to create an open source game engine based on extensions. You can think of them as mods for the programmer. This provides the user with full autonomy and with proprietary software under (almost) no license.
Pololib is a lightwight rendering library focused on simplicity, portability and performance. Inspired by Raylib, this library is still focused on learning graphics programming but also on real application production.
